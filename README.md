# Electrical Engineer Portfolio (Static)

This is a **simple static** portfolio site (no Jekyll, no themes, no blog).

## Quick start on GitHub Pages
1. Create a new repo (or use `username.github.io`).
2. Upload all files from this folder.
3. GitHub → **Settings → Pages**
   - Source: Deploy from a branch
   - Branch: main/master
   - Folder: / (root)
4. Open your site URL.

## What to edit
- `index.html`:
  - Replace `YOUR FULL NAME`, title, and About text
  - Replace links (LinkedIn / Telegram / email / GitHub)
- Replace images in `assets/images/`
  - `hero.jpg` (top banner)
  - `profile.jpg` (your photo)
  - `projectX_Y.jpg` for project images
- Add your resume PDF at `assets/pdf/Resume.pdf`

That's it.
